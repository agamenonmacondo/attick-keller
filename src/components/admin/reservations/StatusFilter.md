# StatusFilter.tsx

- **Que hace**: Barra de tabs para filtrar reservas por estado (Todas, Pendientes, Confirmadas, Sentados, Canceladas) con counts
- **Datos**: Props active, onChange, counts con cantidades por estado
- **Dependencias**: cn
- **Pitfalls**: Los 5 filtros son hardcoded; si se agrega un estado nuevo hay que actualizar FILTERS
