# serviceHours.ts

- **Que hace**: Constantes de horas de servicio del restaurante (apertura, cierre, ultima reserva)
- **Datos**: Exporta SERVICE_START = 12:00, SERVICE_END = 23:00, LAST_RESERVATION = 22:00
- **Dependencias**: Usado por algoritmo de asignacion de mesas y validacion de reservas
- **Pitfalls**: Horas hardcodeadas — si el restaurante cambia horarios, actualizar aqui
